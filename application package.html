Ms Access
Ms Outlook

word processing software

A word processor is a device or software program capable of creating, storing, and printing text documents. It allows users to write and modify text, display it on a screen, save it electronically, and print it out.

functions of word:
Creating, editing, saving and printing documents
Copying, pasting, moving and deleting text within a document
Creating and editing tables
Correcting spelling and grammar
Mail merging

spreadsheets

Spreadsheets are used to store, organize and analyze data. They are used to perform large calculations. Interms of rows and coloumns data are arranged and calculated.

power point

Powerpoint is a part of software package that allows user to create presentations.
It allows user to add images, video, audio and charts in their presentation. It also supports transitions and animations.
