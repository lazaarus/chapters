Programming concepts

To communicate with the computer is to develop a third language- a language that can be understood by both you and the computer. This is what a programming language is, A set of rules that provides a way of instructing the computer to perform certain operations. A computer needs instructions to follow because it cannot think on its own. For instance, when playing a game of solitare you must chose which card to move. These instructions compose only a small part of the solitare program. This program comprises many more instructions that allow a user to perform actions, such as begnining or ending a game, seleecting card's color, and so forth. Therefore, a computer program comprises instructions that direct the actions of the computer.

types of programming language:
1. machine level language
2. assembly language
3. high level language

How does machine level language work?

Machine-level language operates at the hardware level. Here's a simplified overview of how it works:
Fetch: The CPU fetches an instruction from memory.
Decode: The CPU decodes the instruction to determine the operation to perform.
Execute: The CPU executes the instruction, such as adding two numbers or moving data.
Store: The result of the operation is stored in memory or a register.

assembly language

Assembly-level language is a low-level programming language that is closely related to machine language. It uses mnemonics (short, human-readable codes) to represent machine-level instructions. Each mnemonic corresponds to a specific machine instruction, making it easier for programmers to write and understand low-level code.

Characteristics of Assembly Language:
Hardware-Specific: Each CPU architecture has its own assembly language.
Human-Readable: Uses mnemonics like MOV, ADD, and JMP instead of binary.
Direct Control: Provides precise control over hardware resources.
Efficiency: Programs written in assembly language are highly optimized for performance.

high level language

High-level languages are programming languages that are closer to human language and further from machine language. They provide a high level of abstraction from hardware, making them easier to learn and use. Examples include Python, Java, C++, and JavaScript.

Characteristics of high level language

Characteristics of High-Level Languages:
Readability: Syntax is similar to human language.
Portability: Programs can run on different platforms with minimal changes.
Abstraction: Hides low-level details like memory management.
Productivity: Faster development compared to low-level languages.

Programming in C

Programming in C is a fundamental skill for any aspiring programmer. C is a high-level language that is a base for many languages in computer. C is a general-purpose language that can be used for a wide range of applications.

C programming contains different types of variables, operators, control structures, functions, arrays, and pointers. It allows user to create problem solving programs easily. C programming has many keywords that has a different meaning and can't be used in as a variable or a constant.

int num, reversedNum = 0, remainder, originalNum;
printf("Enter a number: ");
scanf("%d", &num);

originalNum = num;
while (num != 0) {
remainder = num % 10;
reversedNum = reversedNum * 10 + remainder;
num /= 10; }
if (originalNum == reversedNum) {
printf("%d is a palindrome.\n", originalNum); }
else {
printf("%d is not a palindrome.\n", originalNum); }
